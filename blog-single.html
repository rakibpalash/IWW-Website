How to Connect OneDrive to Zoho CRM - Complete Setup Guide (2026)

Install Easy OneDrive for Zoho CRM, authenticate Microsoft 365, confirm the default workflows, and configure folder naming plus attachment sync.

By: Insta Web Works Editorial Team | Published: 2026

Overview

The live Insta Web Works guide explains that connecting OneDrive to Zoho CRM starts by installing the extension from Zoho Marketplace, authenticating OneDrive, confirming the default workflows and related list, and then defining your folder structure.

For Microsoft 365 teams, the goal is to keep folders, attachments, and document access inside CRM instead of switching platforms all day.

Step 1: Install the extension

Open Zoho Marketplace, search for Easy OneDrive for Zoho CRM, start the free trial or install the extension, and select the users or profiles who need access.

Step 2: Authenticate OneDrive

From the extension settings page in Zoho CRM, connect OneDrive through the Microsoft authentication flow so the extension can create and manage record-linked folders.

Step 3: Confirm workflows and configure naming

After installation, verify that the default workflows and related list are available. Then configure your naming rules, folder structure, and attachment sync behavior.

Step 4: Test inside a record

Create or open a CRM record, confirm that the linked folder appears, upload a file, and test whether CRM attachments sync to the target OneDrive location correctly.
